========================================
DATABASE SETUP INSTRUCTIONS
========================================

1. Run the main database setup:
   mysql -u root -p dfcts < database.sql

2. Add the police login accounts (all PENDING):
   mysql -u root -p dfcts < police_logins.sql

3. (Optional) Approve 3 test accounts for immediate testing:
   mysql -u root -p dfcts < approve_test_accounts.sql

4. Access the system:
   - Admin: http://localhost/dfcts/admin_dashboard.php
   - Police: http://localhost/dfcts/police_dashboard.php
   - Forensic: http://localhost/dfcts/forensic_dashboard.php
   - Login: http://localhost/dfcts/login.php

TROUBLESHOOTING
========================================

If you get database connection errors:
1. Make sure MySQL server is running
2. Check if the 'dfcts' database exists
3. Verify database credentials in includes/db.php
4. Ensure PDO MySQL extension is enabled in PHP
